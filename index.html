📁 Aktifkan Realtime Database dan gunakan rules berikut: 
{
  "rules": {
    "rooms": {
      "$roomId": {
        ".read": true,
        ".write": false,
        "users": {
          "$userId": { ".read": true, ".write": "!data.exists() || data.exists()" }
        },
        "messages": {
          "$msgId": { ".read": true, ".write": "!data.exists()" }
        }
      }
    }
  }
}
✅ Pastikan databaseURL dan API key sudah benar.
